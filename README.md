## 🔥 ¿Por qué CSSReel?

**CSSReel** no es solo una lista de animaciones; es una herramienta de ingeniería visual. Cada efecto ha sido diseñado para ser **liviano, modular y libre de dependencias**.

- **Editor en Vivo:** Cambia colores, velocidades, tamaños y radios de borde con sliders quirúrgicos.
- **Sin Dependencias:** Todo funciona con código estándar que puedes pegar en cualquier proyecto (React, Vue, HTML estático).
- **Matemáticas Precisas:** Animaciones basadas en cinemática fluida, morphing de bordes y efectos de distorsión digital.

## 🚀 Instalación y Uso

1. **Clona el repo:**
   ```bash
   git clone https://github.com/andrewavilan-work/CSSReel.git
   ```
2. **Abre el archivo:**
   Simplemente haz doble clic en `index.html`. No requiere servidor, compilación ni `npm install`.
